Create a telegram bot without a code

1. Create account(if you don't have).
2. Create a bot in this bot BotFather.
3. Write to the BotFather /newbot.
4. Write name(name must end with Bot).
5. Click to link sent us BotFather.
6. Click in this link ManyBot.
7. Click button "Start".
8. Write command /addbot.
8. Go to BotFather amd copy token.
9. Repeat to ManyBot and click button "I copy token".
10. Send token.
11. Go to you bot and write "/commands".
12. Click button "Create command".
13. Write name command.
14. And send what the command will do and click "Save".

16. Choose command and write name menu item.
17. Bot is ready!
